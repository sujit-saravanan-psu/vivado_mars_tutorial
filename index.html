Setting up a VM on Mac:

  • For students using Mac with a M1 chip (MacBoook air/pro/mini from 2020 onwards or iMac from 2021 onwards):

    • Download UTM from: https://mac.getutm.app/
    • Download Ubuntu for arm from: https://ubuntu.com/download/server/arm
    • Detailed instructions: https://mac.getutm.app/gallery/ubuntu-20-04
    • Installation video demo: https://youtu.be/hnwK-nkXolc

  • For Students on an Intel Mac:

    • Download Virtualbox 6.1.30 from: https://www.virtualbox.org
    • Download Ubuntu from: https://ubuntu.com/download/desktop
    • Installation video demo: https://youtu.be/x5MhydijWmc
NOTE: I recommend allocating 70GB of storage at the very least, as Vivado is a large program

VERY IMPORTANT NOTE: Make sure you are using a Fixed Size hard disk if you're using VirtualBox



How to install Vivado 2019.2 for CMPEN 331(Linux)

(Alexis Dumas approach, don't say a trans never did nothin for ya)

  • Boot into your Ubuntu Virtual Machine, everything below assumes you're in your Virtual Machine
  • Make an account on xilinx.com
  • Download Vivado 2019.2 (the final HLx version that has the WebPACK edition for Linux, which is the only free edition, and the one required by this class) from this link: https://www.xilinx.com/member/forms/download/xef.html?filename=Xilinx_Unified_2019.2_1106_2127_Lin64.bin
  • Start the "Terminal" application

NOTE: You can run commands in the terminal by simply copy pasting them in using CTRL+C to copy the command, then using CTRL+SHIFT+V inside the terminal to paste the command, and then hitting ENTER to run the command
  • Run
    sudo apt install libncurses5-dev libncursesw5-dev libncurses5
    in the terminal
  • Run
    sudo apt install libtinfo5 libtinfo-dev
  • Run
    cd ~/Downloads
    in the terminal
  • Run
    chmod +x Xilinx_Unified_2019.2_1106_2127_Lin64.bin
    in the terminal
  • Run
    sudo mkdir -m 777 ~/Downloads/xilinix
  • Finally, run
    sh Xilinx_Unified_2019.2_1106_2127_Lin64.bin
    in the terminal
  • If a warning pops up, click ok
  • If a prompt asking you to install a newer version pops up, click continue
  • Click next
  • Enter your username and password from before
  • Check all the boxes
  • Select Vivado
  • Select Vivado HL WebPACK
  • Uncheck both UltraScale and UltraScale+ options to make the install faster
  • If you get a "Cannot write to *some file path*. Check read/write permissions" warning, click on the three dots next to the "Select the installation directory text field and navigate to the "home/*username*/Downloads/xilinix" folder and click ok
  • Continue through the rest of the installer
  • It should work now! Just remember, it takes awhile to launch so you might think it's not working



How to install Vivado 2019.2 for CMPEN 331(Windows)

  • Make an account on xilinx.com
  • Download Vivado 2019.2 (the final HLx version that has the WebPACK edition for Windows, which is the only free edition, and the one required by this class) from this link: https://www.xilinx.com/member/forms/download/xef.html?filename=Xilinx_Unified_2019.2_1106_2127_Win64.exe
  • Enter your username and password from before
  • Choose to install Vivado
  • Choose to install the WebPACK version
  • Uncheck both Ultra options to make the install faster
  • Continue through the rest of the installer
  • It should work now! Just remember, it takes awhile to launch so you might think it's not working



Setting up MARS(Linux):

  • Boot into your Ubuntu Virtual Machine
  • Start the "Terminal" application
  • Run the following commands in the terminal:
  • sudo apt install wget
  • sudo apt install openjdk-17-jre
  • sudo wget -P ~/.local/share/applications https://cdn.discordapp.com/attachments/920875012234289246/931236214110044270/MARS.desktop
  • sudo wget -P ~/.local/share/mars https://cdn.discordapp.com/attachments/920875012234289246/930995668670091304/Mars4_5.jar
  • The MARS application should now be inside your application menu
